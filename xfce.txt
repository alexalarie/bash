To create a new icon in the applications menu on xfce (tested with xfce version 4.12)

Add this file to /home/alex/.local/share/applications/

[Desktop Entry]
Version=57.0
Type=Application
Name=Firefox Quantum
Exec=/opt/firefox/57.0/firefox
Icon=/opt/firefox/57.0/browser/icons/mozicon128.png
Categories=Network; 

Sub-Menu      Categories
--------      ----------
Accessories 	Utility
Development 	Development
Games 	      Game
Graphics 	    Graphics
Internet 	    Network
Multimedia 	  AudioVideo
Office 	      Office
System 	      System

For Tor Web Browser
-------------------

[Desktop Entry]
Version=64-7.5
Type=Application
Name=Tor Browser
Exec=/opt/tor/64-7.5/start-tor-browser 
Exec=sh -c '"/home/alex/Desktop/tor-browser_en-US/Browser/start-tor-browser" --detach 
X-TorBrowser-ExecShell=./Browser/start-tor-browser --detach
Icon=/opt/tor/64-7.5/Browser/browser/icons/mozicon128.png
Categories=Network;WebBrowser;Security; 
StartupWMClass=Tor Browser
